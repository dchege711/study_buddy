(static) update(savedCards, metadataQuery, attributeName) → {Promise}

Update the metadata with the new cards' details. This method is usually called by CardsMongoDB.update(). An array is needed to prevent race conditions when updating metadata from more than one card.

Parameters:
Name Type Description
savedCards Array

Array of cards
metadataQuery JSON

An identifier for the metadata document. This argument was added in order to update the global public user account. If not specified, it defaults to the owner of the first card in savedCards.
attributeName String
