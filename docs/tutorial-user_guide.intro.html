Wiki: About Flashcards by c13u

I wrote this application mainly for me. In college, I was a huge fan of dividing my notebook into two columns, with the right column being three times as wide as the left one. In the right column, I'd have class notes. In the thinner left column, I'd have questions that cover the content on the right. Revising for exams was easy, cover up the right side and try answering the questions on the left.

This app is an attempt to recreate that behavior in software - I have a bunch of notes that I'd want to review periodically. There is established software out there on flashcards, e.g. Anki, SuperMemo, Quizlet, etc. They have their upsides.

I prefer my own app mainly because I design it as I see fit. It's a side project that I am proud to share with the world. I also avail the code that runs the website in the hope that some of you are interested in peeking under the hood too.

Feel free to send me bugs, feature requests and feedback, either on Github or at d.chege711(at)gmail[dot]com.
